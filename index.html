Code Test - Ad Tech Engineer

Below is a short code test for you to show off your ability as an Ad Tech Engineer. Feel free to add in any customizations beyond the stated requirements.

Requirements

Create a StickyBanner Object that creates a Banner Ad fixed to the top or bottom of the viewport. This Ad should remain fixed to the top or bottom of the viewport when a user scrolls this page.

The StickyBanner object should have at least the following functionality:

  1. Can create a new StickyBanner instance by calling new StickyBanner(html, config). The html argument is a string of the html to be placed in the Banner Ad. The config argument is an object that recognizes two keys:
    • position - accepts string: 'top' or 'bottom' - determines whether to fix the Banner Ad to the bottom or top of the viewport
    • hidingEnabled - accepts boolean: true or false - determines whether the Banner Ad should hide when blockers are visible (more on this later!)
  2. config values should be set to defaults if not specified in the construtor:
    • position = 'top'
    • hidingEnabled = false
  3. Can destroy a StickyBanner instance by calling a destroy method.
  4. Multiple instances of a StickyBanner can exist at a time, but only one at each position. In this scenario we can have, at most, one 'top' StickyBanner and one 'bottom' StickyBanner active at the same time.
  5. When creating a new StickyBanner of the same position as an active StickyBanner, the older instance should be destroyed.
  6. if hidingEnabled is set in a StickyBanner's config, the Banner Ad should hide when a "blocker" element is in view. For this test, blocker elements all have the class "blocker" on them. There are 3 on this page and are colored red with a height of 300px.

This should be done using only pure Javascript (no external libraries, jQuery, etc.), and only by creating a 'sticky-banner.js' file (no editing this test page). You can see some examples of the StickyBanner usage in a script element at the bottom of the page.

You can assume the Banner Ad will be 320px in width and 50px in height and that the user will be using a mobile device to view this page (don't worry about Tablet or Desktop support) with a modern browser.

If you have any questions, feel free to email me at andy@kargo.com. Thanks, and Good Luck!!
